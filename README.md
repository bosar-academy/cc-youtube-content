# YouTube Content Skills

> 6 Claude Code skills for YouTube content creation — research, scripting, titles, thumbnails, and analytics

## Skills Included

| Skill | Description |
|-------|-------------|
| `write-script` | Write YouTube video scripts with research and hooks |
| `research-brief` | Generate YouTube video research briefs |
| `generate-title` | Generate optimized YouTube video titles |
| `youtube-thumbnail` | Generate YouTube thumbnails with Gemini AI |
| `youtube-analytics` | Analyze YouTube channel performance |
| `cross-niche-outliers` | Find viral content patterns across YouTube niches |

## Installation

```bash
git clone https://github.com/bohdan-aif/cc-youtube-content.git
cd cc-youtube-content
./install.sh --all
```

## Required API Keys

- `YOUTUBE_API_KEY` — YouTube Data API
- `GEMINI_API_KEY` — Thumbnail generation
